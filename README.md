# v-em-00

### Running tests
Clone the project & run ```mvn test``` command in project directory.

### Building an application
Clone the project & run ```mvn clean install``` command in project directory. Executable JAR application is going to be in **/target** directory, named **v-em-00-1.0.0-exec.jar**.

### Running application
Run ```java -jar v-em-00-1.0.0-exec.jar "<PATH_TO_INPUT_FILE>"``` command i.e. ```java -jar v-em-00-1.0.0-exec.jar "/Users/user/input.txt"```
